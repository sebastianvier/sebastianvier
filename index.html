Credit Card Project

The aim of the models here is to predict whether a customer will accept or not an offered credit card. The data is very unbalanced, so the real challenge is to get as much 'yes' recall, maintaining a low number of false positives. For this project a 7-steps function was built in order to pre-process the data.
